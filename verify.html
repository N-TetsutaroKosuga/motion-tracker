対象ボーン（8本、poseWorldLandmarks のメートル座標で算出）: upperArmL(11-13) / upperArmR(12-14) / forearmL(13-15) / forearmR(14-16) / thighL(23-25) / thighR(24-26) / shinL(25-27) / shinR(26-28)。 剛体である骨の長さは本来フレーム間で一定のはずなので、変動係数(CV=標準偏差/平均)が トラッキング精度の指標になる。左右対称性は対になる4組（上腕/前腕/大腿/下腿）の平均長の相対差。 ジッターσは「静止判定（2D速度 < 0.25 /秒 相当）」中のフレームのみを対象にした位置のばらつき、 RMSは検出フレーム間の位置変化のRMS（静止・動作を問わない）。関節可動域逸脱は、肘/膝の3点角度が 10°未満（ランドマーク重なり＝オクルージョン崩壊の疑い）、または前フレームから45°/フレーム超で 急変した場合（トラッキング破綻の疑い）を累積カウントしたもの。

ブラウザ内で完結（映像は外部に送信されません）。動かすには python3 -m http.server 8000 でこのフォルダを配信し、 http://localhost:8000/verify.html を Chrome / Safari で開いてください。 実カメラ・フェイクカメラ（--use-fake-device-for-media-stream）のどちらでも同一の経路で動作します。
